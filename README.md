# Student project


## Quick start
`git clone https://github.com/PritulaRoman/student.git`

### Install dependencies
`npm install npm@latest -g`

From students dir:
`npm install`


### Build JS
From students dir:

One time run
`npm run biuld`

Run with watcher
`npm run dev`
